# BibleMarker Wiki

Welcome to the **BibleMarker** wiki—your guide to getting started and using the app for deep Bible study with the Inductive Bible Study Method.

## What is BibleMarker?

BibleMarker is a cross-platform **desktop** Bible study application for deep text study. Mark, annotate, and analyze Scripture with powerful study tools. All data is stored locally (no cloud required). It is distributed as a desktop app only (macOS, Windows, Linux)—no web or PWA version at this time.
